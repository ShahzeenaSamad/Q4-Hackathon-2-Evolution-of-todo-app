# Claude Code Skills

This directory contains reusable agent skills for spec-driven development.

## Available Skills

### 1. fastapi-backend-implementer
Implements FastAPI backend features based on approved specifications.

### 2. frontend-nextjs-builder
Builds Next.js 16+ frontend components and pages using App Router.

### 3. qa-testing-validator
Validates implementations against specifications and runs tests.

## Usage

Skills are automatically invoked by Claude Code based on task requirements.

## Skill Development

To create a new skill:
1. Create a new markdown file in this directory
2. Follow the skill template structure
3. Register the skill in Claude Code settings
